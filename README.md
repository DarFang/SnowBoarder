- triggers and collisions
- effector
- torque
### Design Overview:
- Casual game, move the snowboarder forward

## Game Mechanics:
- When on ground, go faster, add dust effect
- allow for player to control the rotation
- Add reset level mechanic when hit on head, or finish line
